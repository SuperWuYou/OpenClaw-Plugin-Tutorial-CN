## 10.3 开发工具

### 10.3.1 CLI 命令

```bash
# 插件管理
openclaw plugins list                    # 列出所有插件
openclawaw plugins info <id>            # 查看插件详情
openclaw plugins install <package>       # 安装插件
openclaw plugins enable <id>             # 启用插件
openclaw plugins disable <id>            # 禁用插件
openclaw plugins doctor                  # 检查插件健康状态

# 调试
openclaw gateway --verbose               # 详细日志模式
openclaw logs                            # 查看日志
```

### 10.3.2 开发环境设置

```bash
# 1. 克隆仓库
git clone https://github.com/openclaw/openclaw.git
cd openclaw

# 2. 安装依赖
pnpm install

# 3. 构建项目
pnpm build

# 4. 开发模式运行
pnpm dev

# 5. 运行测试
pnpm test

# 6. 测试单个插件
pnpm test extensions/my-plugin
```

### 10.5.2 代码规范

```typescript

// 2. 遵循 ESLint 规则

// 4. 编写单元测试

/**
 * 示例工具函数

 * @returns 处理结果

  // 处理逻辑

// 测试
describe("exampleFunction", () => {
```

### 10.5.3 提交消息格式

```
<type>(<scope>): <subject>

<body>

<footer>
```

**类型 (type)：**
- `feat`: 新功能
- `fix`: Bug 修复
- `docs`: 文档更新
- `style`: 代码格式
- `refactor`: 代码重构
- `test`: 测试相关
- `chore`: 构建/工具

**示例：**
```
feat(plugins): add new channel plugin for Discord

- Implement Discord gateway integration
- Add message send/receive handlers
- Support thread and reaction features

Closes #123
```

## 10.7 插件安全

### 10.7.1 安全最佳实践

1. **不要在代码中硬编码密钥**
   ```typescript
   // ❌ 不推荐
   const apiKey = "sk-hardcoded-key";

   // ✅ 推荐
   const apiKey = config.apiKey || process.env.MY_API_KEY;
   ```

2. **验证所有外部输入**
   ```typescript
   function validateInput(input: unknown): string {
     if (typeof input !== "string") {
       throw new Error("Invalid input type");
     }
     // 进一步验证
     if (input.length > 1000) {
       throw new Error("Input too long");
     }
     return input;
   }
   ```

3. **使用 HTTPS**
   ```typescript
   // 始终使用 HTTPS 进行 API 调用
   const response = await fetch("https://api.example.com/endpoint");
   ```

4. **实现速率限制**
   ```typescript
   const rateLimiter = new Map<string, number[]>();

   function checkRateLimit(key: string, limit: number, window: number): boolean {
     const now = Date.now();
     const requests = rateLimiter.get(key) || [];

     // 清理过期请求
     const validRequests = requests.filter((t) => now - t < window);

     if (validRequests.length >= limit) {
       return false;
     }

     validRequests.push(now);
     rateLimiter.set(key, validRequests);
     return true;
   }
   ```

### 10.7.2 敏感数据处理

```typescript
// 标记敏感配置字段
const configSchema = {
  uiHints: {
    apiKey: { sensitive: true },
    password: { sensitive: true },
  },
};

// 日志中过滤敏感信息
function sanitizeForLog(obj: unknown): unknown {
  if (typeof obj !== "object" || obj === null) return obj;

  const result: Record<string, unknown> = {};
  const sensitiveKeys = ["apiKey", "password", "secret", "token"];

  for (const [key, value] of Object.entries(obj as Record<string, unknown>)) {
    if (sensitiveKeys.some((k) => key.toLowerCase().includes(k))) {
      result[key] = "***REDACTED***";
    } else {
      result[key] = value;
    }
  }

  return result;
}
```
