# 第十一章：更新日志

本章记录插件开发指南和插件系统的重要更新历史。

## 11.1 插件系统版本历史

### 11.1.1 版本 2026.1 (当前版本)

**新功能：**

- 添加新的生命周期钩子 API `api.on()`
- 支持插件槽位（slots）用于互斥插件
- 改进配置验证和错误提示
- 添加 Provider 插件类型的 OAuth 刷新支持

**改进：**

- 优化插件加载性能
- 改进日志系统结构化输出
- 增强 HTTP 路由的安全性检查

**修复：**

- 修复插件配置热重载问题
- 修复钩子优先级排序错误
- 修复内存泄漏问题

### 11.1.2 版本 2025.12

**新功能：**

- 添加自定义命令注册 `api.registerCommand()`
- 支持 JSON Schema 验证配置
- 添加 UI 提示（uiHints）支持

**改进：**

- 重构插件 SDK 模块化结构
- 改进类型定义完整性
- 优化 Gateway RPC 性能

### 11.1.3 版本 2025.11

**新功能：**

- 添加子代理相关钩子
- 支持工具结果持久化钩子
- 添加消息写入前钩子

**改进：**

- 改进错误处理和错误消息
- 优化钩子执行性能
- 增强日志详细程度控制

### 11.1.4 版本 2025.10

**新功能：**

- 初始插件系统架构
- 支持工具、钩子、HTTP 路由注册
- 基础配置系统

## 11.6 迁移指南

### 11.6.1 从 2025.12 迁移到 2026.1

**使用新的钩子 API：**

```typescript
// 旧 API（仍然支持）
api.registerHook("message_received", handler, {
  name: "my-plugin.message-handler",
});

// 新 API（推荐）
api.on("message_received", handler, { priority: 10 });
```

**使用插件槽位：**

```json5
// 配置示例
{
  plugins: {
    slots: {
      memory: "memory-lancedb", // 或 "memory-core" 或 "none"
    },
  },
}
```

### 11.6.2 从 2025.11 迁移到 2025.12

**添加 UI 提示：**

```json
{
  "id": "my-plugin",
  "configSchema": {
    "type": "object",
    "properties": {
      "apiKey": { "type": "string" }
    }
  },
  "uiHints": {
    "apiKey": {
      "label": "API 密钥",
      "sensitive": true
    }
  }
}
```
